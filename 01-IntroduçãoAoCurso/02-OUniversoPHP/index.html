O universo PHP

Historia

É uma linguagem interpretada livre que atua do lado do servidor, tem seu melhor uso no desenvolvimento de aplicações web dinamicas.

Figura entre as primeiras linguagens passiveis de inserção de documentos HTML e é uma linguagem extremamente modularizada. O que a torna ideal para servidores web

O PHP é utilizado em aplicaçõescomo o Facebook, Drupal, Joomla, WordPress, Magento, Oscommerce, Laravel, Synfony, Zend Framework e muitos outros.

  • 1994/1995
    PHP 1 e 2

    • O PHP é criado por asmus para ser usado no desenvolvimento de sua pagina pessoal,
    era apenas um conjunto de scripts que foi chamado de Personal Home Page Tools.

  • 1997/1998
    PHP 3

    • Andi e Zeev entraram no time para reescrever a base do PHP como uma linguagem
    completa de programação e varios desenvolvedores passam a colaborar com o desenvolvimento
    do PHP

  • 2000/2003
    PHP 4

    • A versão 3 é totalmente abandonada e o 4 é desenvolvido com recursos de OO. O que atraiu milhares
    de desenvolvedores para a linguagem.

  • 2004/2014
    PHP 5

    • Ocorre a acensão do php com a versão 2 da zend engine, agora totalmente orientado a objetos, banco de dados com PDO, JSON nativo, namespaces, interfaces, traits e diversos outros recursos que solidificaram o PHP.

  • 2015/20XX
    PHP 7

    • Com performance surpreendente e fortificação em recursos de OO , o PHP é visto como uma linguagem madura presente em mais de 93.5% dos sites na internet sendo a mais utilizada linguagem de programação do mundo.
